## CSV Column Mapping

### Your CSV:

```
name,email,completion_date,score
John Doe,john@example.com,2025-10-17,95
Jane Smith,jane@example.com,2025-10-18,98
```

### You'll see buttons:

```
[name] [email] [completion_date] [score]
```

### Click [name] on text element:

```
Text changes from: "Double-click to edit"
                to: "{{name}}"
```

### Click [completion_date] on another text:

```
Text changes to: "{{completion_date}}"
```

### In generated PDF:

```
John Doe's certificate shows:
- Name: "John Doe"
- Date: "2025-10-17"
```
