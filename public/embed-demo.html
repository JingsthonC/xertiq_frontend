üíª Integration Examples

Simple iframe embed that shows the verification form:

<!-- Basic embed - shows search form --> <iframe src="https://your-domain.com/embed/verify" width="100%" height="600" frameborder="0" style="border: 1px solid #e0e0e0; border-radius: 12px;" title="XertiQ Document Verification" ></iframe>

Auto-verify a specific document by passing the document ID:

<!-- Auto-verify specific document --> <iframe src="https://your-domain.com/embed/verify?doc=cert_certificate_John_Doe_123_abc" width="100%" height="700" frameborder="0" style="border: 1px solid #e0e0e0; border-radius: 12px;" title="Certificate Verification" ></iframe>

Fully responsive embed that maintains aspect ratio:

<div style="position: relative; width: 100%; max-width: 800px; margin: 0 auto;"> <div style="position: relative; padding-bottom: 75%; height: 0; overflow: hidden;"> <iframe src="https://your-domain.com/embed/verify?doc=YOUR_DOC_ID" style="position: absolute; top: 0; left: 0; width: 100%; height: 100%; border-radius: 12px;" frameborder="0" title="XertiQ Verification" ></iframe> </div> </div>

React component for easy integration:

import React from 'react'; const VerificationWidget = ({ documentId }) => { const embedUrl = documentId ? `https://your-domain.com/embed/verify?doc=${documentId}` : 'https://your-domain.com/embed/verify'; return ( <div className="verification-widget"> <iframe src={embedUrl} width="100%" height="650" frameBorder="0" style={{ border: '1px solid #e0e0e0', borderRadius: '12px', boxShadow: '0 2px 8px rgba(0,0,0,0.1)' }} title="Document Verification" /> </div> ); }; export default VerificationWidget;

üìñ Quick Start Guide

  1. Copy one of the code examples above
  2. Replace your-domain.com with your XertiQ instance URL
  3. Optionally add a document ID to the URL parameter: ?doc=YOUR_DOC_ID
  4. Paste the code into your website HTML
  5. The widget will load automatically with "Powered by XertiQ" branding
